Decorator Pattern

An alternative to subclassing, this modifies the properties of objects without recreating them. Both of the decorator functions below accepts a hero object and returns a "decorated object". When the base Hero class activates Power Armor, it adds an attribute (flySpeed) and a function (flyMove); when activating Suit AI, it adds a new attribute (foresight) and modifies an existing attribute (baseSpeed).
